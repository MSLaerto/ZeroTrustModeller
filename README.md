# ZeroTrustModeller
Программа для автоматизированного анализа IoT-инфраструктуры и генерации планов внедрения модели Zero Trust.

## Возможности
1. Инвентаризация IoT-устройств
2. Оценка уязвимостей и рисков
3. Генерация пошагового плана внедрения Zero Trust
4. Адаптация под российские стандарты (ФСТЭК, ГОСТ)

## Установка
1. Установите Python 3.8+
2. Клонируйте репозиторий
3. Установите зависимости: `pip install -r requirements.txt`

## Запуск
`python main.py`


## Советы по использованию
1. Внести крайне подробно все устройства
2. Подробно описать сеть, предварительно разбив устройства в ней по группам взаимодействия
3. Данные об ифраструктуре сохраняются в data/infrastructure.json
4. Отчеты сохраняются в reports/

## Структура проекта
- `main.py` - точка входа
- `modules/` - модули программы
- `config/` - конфигурационные файлы
- `data/` - данные пользователя
